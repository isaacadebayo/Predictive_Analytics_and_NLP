This repository constitute all models fine tuned or indexed from public datasets

* Notebook found in the dataset include image generation, model, fine tuning, multimodal chatbot and predictive analytics
* Algorithms implemented are from tensor flow, pytorch, sklearn, pandas, numpy, seaborn and matplotlib
* Datasets were derived from Kaggle database and Hugging Face

* Use of different transformer algorithm and different types of diffusion transformers.
* **Deep learning occurs using Keras and PyTorch** : Visual transformers (ViT), Audio transformers (OpenAI Whisper), Convolutional Neuraal Networks (CNNs), BERT aarchitecutres
* **Deployment and Maturity are seen in my hugging face space and Gradio UI**
* **Generative AI & Agentic Systems** using RAG, autonomous agents, vector databases, prompt engineering .. etc.
  
* Code debuggings was assisted using gemini and claude
* Work and future works are hosted on hugging face
